CIA Part 1 Practice

Question 1:

Who is responsible for assessing individual objectivity of internal auditors?

  • A. The chief executive officer.
  • B. The board.
  • C. The audit committee.
  • D. The chief audit executive.

Answer: The chief audit executive.

Question 2:

An internal auditor becomes aware of illegal activity but takes no action. What ethical code is violated?

  • A. Did not violate the Code of Ethics.
  • B. Violated the Code of Ethics by failing to protect the well-being of the general public.
  • C. Violated the Code of Ethics by knowingly becoming a party to an illegal act.
  • D. Did not violate the Code of Ethics as conclusive information was not gathered.

Answer: Violated the Code of Ethics by knowingly becoming a party to an illegal act.

Question 3:

What should an internal auditor do if offered a gift of moderate value during the holiday season?

  • A. Accept it regardless of circumstances.
  • B. Not accept it prior to submission of the final engagement communication.
  • C. Not accept it if it is presumed to impair the auditor's judgment.
  • D. Not accept it, regardless of circumstances.

Answer: Not accept it if it is presumed to impair the auditor's judgment.

Question 4:

Which type of standards applies to the characteristics of internal audit service providers?

  • A. Implementation Standards.
  • B. Performance Standards.
  • C. Attribute Standards.
  • D. Independence Standards.

Answer: Attribute Standards.
